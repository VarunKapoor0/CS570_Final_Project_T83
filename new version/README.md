## Running the Programs

Before running the scripts for the first time, give execution permission:

chmod +x basic.sh
chmod +x efficient.sh

Then you can run the programs using:

./basic.sh <input_file> <output_file>
./efficient.sh <input_file> <output_file>

Example:
./basic.sh in1.txt out_basic1.txt
./efficient.sh in1.txt out_efficient1.txt

Output format:
1.cost（168）
2.aligned X
3.aligned Y
4.time ms
5.memory KB
